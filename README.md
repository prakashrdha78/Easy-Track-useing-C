# EasyTrack – Daily Expense Tracker

## 📌 Project Overview
EasyTrack is a Full Stack Daily Expense Tracker project created using HTML, CSS, JavaScript, Node.js, Express, and MongoDB.

The project helps users:
- Add daily expenses
- View expense history
- Delete expenses
- Manage spending records
- Store data using localStorage or MongoDB backend

This project was created as a university project.

# 🚀 How to Run the Project

## Frontend Only

1. Open the project folder in VS Code
2. Open `index.html`
3. Run using Live Server

---

## Full Stack Version

### Install Dependencies

```bash
npm init -y
npm install express mongoose cors dotenv
```

### Run Server

```bash
node server.js
```
